# purescript-truffle-pet-shop

[Truffle's Pet Shop tutorial](http://truffleframework.com/tutorials/pet-shop) ported to [`PureScript`](http://www.purescript.org/).


# Installation

- Install dependencies (only once)

```bash
yarn
```

- Build PS sources
```
pulp build
```

- Run blockchain locally
```
truffle develop
```

- Serve app
```
yarn start
```
